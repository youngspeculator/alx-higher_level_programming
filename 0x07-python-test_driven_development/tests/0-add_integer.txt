This is a module which defines an integer addition function 'add_integer(a, b=98)'
```add_integer()``` will return the addition of the two args.
                Mathematically, it is equivalent to the `+` operand

Import the function from module:
        >>> add_integer = __import__('0-add_integer').add_integer

Usage:

Testing two positive integers
        >>> add_integer = __import__('0-add_integer').add_integer
        >>> print(add_integer(1, 2))
        3

Testing heterogenous integers (positive and negative)
        >>> add_integer = __import__('0-add_integer').add_integer
        >>> print(add_integer(100, -2))
        98

Testing floating point values
        >>> add_integer = __import__('0-add_integer').add_integer
        >>> print(add_integer(6.0, 2.0))
        8

Floating and non-floating point values
        >>> add_integer = __import__('0-add_integer').add_integer
        >>> print(add_integer(2.3, -5))
        -3
::

Testing one integer

WHAT ABOUT non-numbers
```add_integer()``` requires that both args are either ints or floats
if this requirement is not met, i.e either arg is a non-integer or a non-float, a TypeError is raised
